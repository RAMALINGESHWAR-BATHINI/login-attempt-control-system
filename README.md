# login-attempt-control-system
Python login security system with limited attempts and temporary access blocking.

## Overview
This project demonstrates a secure login system with limited password attempts.

The system blocks access temporarily after multiple incorrect login attempts to prevent unauthorized access.


## Features
- Username and password validation
- Limited login attempts
- Temporary lock system
- Basic authentication flow


## Technologies Used
- Python
- Time Module


## How to Run

```bash
python main.py
